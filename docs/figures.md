# Figures

This file records reproducible figures used by the paper and preview build.

## Evaluation Evidence Flow

- Generator: `scripts/render_evaluation_figure.py`
- Paper source: `paper/figures/evaluation_evidence_flow.tex`
- Preview image: `output/pdf/figures/evaluation_evidence_flow.png`
- Purpose: show how generated HDL, sweep, fuzz, and Yosys outputs pass trust checks before supporting the paper's headline claims.

The figure is generated locally with Python and Pillow. It does not require Figma, a paid design tool, or a manual export step.
